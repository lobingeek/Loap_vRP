+ Available commands:

> /tpcds <x,y,z> (Example: /tpcds -1234.00,1234.00,-50)
>
> /nc (On/Off)
>
> /car <model> (Example: /car blista )
>
> /fix (fixes the vehicle)
>
> /tuning (Apply modifications to your vehicle)
>
> /tpway (Teleport to a place marked on the map)
